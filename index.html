    BETA 0.97.1.0 Notes:

    Release Date: 01-01-2024



    Changes:

    • General Visual polishing
    • Added icons for 'Auto block' and 'Auto Install' that will appear or disappear between the main view area and Console area
    • Researched Format nods will now glow in gold
    • Extended main Tutorial pop-up messages to include an explanation about the Effect viewer
    • Improved First time Idling Tutorial pop-up messages to be more informative and clear
    • Click on Effect viewer to toggle between pinning effect list or not, this helps when you want to scroll the list
    • Changed auto calculation for Heat to be stable, now when Heat is in overheating/overcooling state, it will till you exactly how much Heat or Cooling you need to stabilize it when you hover-over Heat icon, no need to do math anymore!
    • Improved performance of 'Auto Install'
    • Improved performance of 'Install All' softwares button in 'System Manager > Auto', it is way less laggy now, and added 'Insufficient credit/Not enough Memory space' messages

    Fixes:

    • Fixed Format nodes 'Increase base Materials production' for all types not giving any effects
    • Fixed Auto block and Install Buttons text showing reversed state
    • Fixed new software glow effect might not turn off as expected
    • Fixed Hardwares and software icons brightness not updating instantly when Credit is insufficient or not
    • Fixed 'Achievement's condition to unlock' tooltip not disappearing when there is new messages in the Console
    • Fixed Filtering, Analysis, Sorting items details not updating in 'Store > Details' list

    Patch BETA 0.97.0.0 Notes:

    Release Date: 27-12-2023



    Changes:

    • Completely new UI
    • Some visual have been removed and some have been repositioned
    • Green, Red, and Blue System's themes have been adjusted to fit the new UI
    • Updated all pop-up tutorials texts to fit the new style

      • G.A.I.A will sabotage the System for X minutes and X effect power, depending on Hack difficulty and G.A.I.A difficulty
      • A random sabotage effect will be applied, eicher Reducing CPS, DPS, CPC, MPM, Energy, or Network

      • New objective: reach the goal in the Lower right corner before times up
      • Hitting Walls or Seekers will end the HACK instantly resulting in being Hacked
    • New Tasks system:
      • Adjusted All three tasks
      • Now all Tasks can be played on Three difficulties, Easy, Normal, and Hard
      • See Balancing section for more information
    • Changed some names and fixed some text typos
    • General performance improvements

    Balancing:

    • Adjusted G.A.I.A Hack attempts intervelas, HACKS are less frequent in general
    • All three Tasks now have the same software effects, it is up to your preference to choose to invest and upgrade:
      • Pressing, Addition, and Sequence Credit multipliers are changed to increase base Credit gain instead of increasing generally
      • Pressing, Addition, and Sequence rewards multiplers are changed to scale with current CPS and DPS for each keyboard instead of increasing by fixed number, this way tasks rewards will always be meaningful and engaging!
    • Increased all Auto clicker effect node in format tree from +2% -> +3%
    • Increased Network listener effect node in format tree from +0.1% -> +0.2%
    • Changed Presser, Adder, and Sequencer node in Format tree to be the same prices, now, you will not invest in Task which you do not like because of price difference
    • Changed 'Click' software Clicktinuum effect to increase click by +0.01% CPS for each Mouse instead of fixed number
    • Increased almost all Online items effect by at least x3 depending on the item
    • Decreased all Advanced Decoding algorithms software prices and condition to unlock

    Fixes:

    • With the introduction of the new UI, all old bugs related to menus such as unable to resize, reposition, or contents out of boundaries, all are now irrelevant to this new UI
    • With the introduction of the new effects system, fixed Crown coins animation and effects gets buggy if Crown coin taken by normal method then from Gift AD sequanthally
    • Fixed a bug where some Timers were not terminated correctly, resulting in some events to run in the wrong time, or overlap with other events
    • Fixed a bug where you lose your Data when researching but not Formatting the System, this bug is not 100% confirmed, please send feedback if it is still present
